Newbie Mode

Fake Sms

This option allows you to send a wap push sms in PDU format. This will show up on the users device as if it came from whatever name you put in the header. So if you entered Update a message will appear in the users sms inbox from Update. Your number will not be displayed to the user, headers are limited to 18 characters and the body is limited to 88 characters. Very good for pranking your friends;-D

Ping

This option allows you to send a silent text message to a users phone and it will alert you if the phone is online. The user will not see this message on their phone.

Custom Bytes

This option allows you to send your own payload in PDU format. Requires advanced knowledge of building PDU sms and limited compared to Expert Mode
------------------------

Expert Mode

This section gives you full access to the payload that is getting sent. Google wap push PDU sms if you want to find information on how to build a particular type of wap push msg.

- and spaces will be removed from the hex payload and are only there to make payload more readable.

Hacker Mode

Warning : requires rooted device
You will not get a delivery report for these messages when sent. you can use adb to view the logcat of your device to see what is sent/received

make sure usb debugging is enabled on your device and you have adb tools installed then use the following commands
adb shell
logcat -b radio

Note about which port the use in /dev folder, a lot of android phones use ports labeled smd so you will have to connect your device by usb and use adb to list the ports

ls /dev |grep smd

Some of these ports will not work and you will have to try a lot of them to see will they work. What I do is go to expert mode and send a normal type sms to my own number then I know that this port is working. Or if you go to AT section and open a port then select ATI and send, if you get back a response this port is working.

Flash

This option allows you to send a flash sms this will display on the users phone in a message dialog (Sender will be shown on some devices)

Voice Mail

This option allows you to send a Voice Mail Waiting Indication message to the users phone. It doesn't really do anything but annoying to the user because they cannot remove the icon

Normal

Just sends a normal sms through the modem

Type 0 or Type Zero

This sms type is used by the certain organisations to keep a users device active for tracking purposes and cannot be seen by the user.

Send Raw PDU String

Probably the most powerful option if you know how to craft a raw PDU sms. In theory you can send any type of sms using this option and if you are a security researcher this would be your tool of choice to find vulnerabilities. This is a PDU in raw format so the number you selected has nothing to do with this option. Google search is your friend for this option.

AT Mode

This section is not related to SMS but allows you to send any AT command to the modem. This is called Hacker mode because users that use tools like gr-gsm and airprobe need an Encryption Key to decrypt GSM traffic which can be then viewed in wireshark. Getting the Encryption Key is very hit and miss and I've managed to get it on a Sony ST26i. Again I repeat these sections are only for advanced users that know what they are doing and what commands to use.
